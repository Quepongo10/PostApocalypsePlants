# Post Apocalipse Plants (Nombre Provisional)

### Web art

Eres transportado a este mundo en el que las plantas se han extinguido por completo de su hábitat natural, dejando atrás paisajes vacíos y una sensación constante de pérdida. En este entorno artificial, las personas como tú observan estas cámaras como una forma de evasión, buscando calma y consuelo al contemplar fragmentos de lo que una vez fue vida.<br>
 Mirar estas grabaciones no solo es un acto de relajación, sino también una forma de conexión emocional con la naturaleza perdida, un intento de recuperar, aunque sea por unos instantes, la serenidad que ofrecían aquellos espacios naturales hoy desaparecidos.
